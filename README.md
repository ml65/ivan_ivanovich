# ivan_ivanovich

## ЗАДАНИЕ:

Иван Иванович работает пилотом. Ровно в полдень 1 января 2020 года он вылетел из Москвы и с тех пор всю жизнь живет по одному и тому же графику:

* он за 2 часа совершает полет на запад на 3 часовых пояса.
* 6 часов отдыхает

Затем цикл повторяется, т.е. снова перелет и отдых, перелет и отдых, и т.д.
Иван Иванович никогда не спит, потому что он отличный пилот.

Напишите функцию на любом языке, которая принимает в качестве параметра номер года и возвращает значение, сколько в Москве будет времени, когда Иван Иванович отпразднует наступление того года, который был передан в качестве параметра. Если новый год застает Ивана Ивановича в полете — он празднует его сразу после приземления, потому что он отличный пилот.

### Примечание:

1. Для упрощения следует считать что существует всего 24 часовых пояса

2. При желании задачу можно решать на любом, даже мертвом языке программирования.

## РЕШЕНИЕ:

Задача имеет неоднозначную постановку. Но, судя по самой задаче, имеется в виду празднование Иваном Ивановичем Нового года по местному часовому поясу, где в данный момент находится Иван Иванович. 

Возможны несколько вариантов решения задачи:

1. Итерационный перебором циклов Иван Ивановича и с поиском события наступления нужного нового года

2. Учитывая, что цикл жизни Иван Ивановича 8 часов и это число кратно продолжительности суток, то можно сделать упрощенный алгорим вычисления события наступления Нового года по местному времени Иван Ивановча.

Первый вариант - более универсальный и он будет работать для разных параметров циклов жизни Ивана Ивановича. Второй вариант требует меньше вычислительных ресурсов и работает быстрее. Но жестко привязан к выданным условиям в ТЗ.

Первый  вариант реализован в двух версиях: simple - только функция, solid - реализаця в виде проекта с возможностью дальнейшего развития, поддержки и использования. В примерах вычисляется время выполнения каждого варианта. 

## УСТАНОВКА И ИСПОЛЬЗОВАНИЕ:

### Вариант simple:
Просто запустите `index.php`:
```bash
php var1/simple/index.php
# или
php var2/index.php
```

### Вариант solid:
Перед использованием необходимо установить зависимости через Composer:
```bash
cd var1/solid
composer install
```

После установки зависимостей можно:
- Запустить основной скрипт: `php index.php`
- Запустить тесты: `./vendor/bin/phpunit` или `php vendor/bin/phpunit`
